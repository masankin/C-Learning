顺序容器（序列容器）
=========
- vector
- list
- deque

##公共成员函数
###push_back(t),pop_back()
在容器末尾，增加t的副本及删除列尾数据。  
注意`pop_back（）`操作返回void。
###front(),back()
返回列首和列尾数据的引用或常量引用。
###insert()
|重载|说明
|----|-----
|insert(p, t)|在迭代器p之前位置插入数据t的副本
|inset(p, n, t)|在迭代器p之前位置插入t的n个副本
|insert(p, i, j)|在p之前插入**[i, j)**内的所有数据副本。

注意：第3种重载，要求i，j不能是该容器的迭代器。  
换句话说，是把其他容器内的数据，插入进来。
```cpp
int a[] = {1, 2, 3, 4, 5}
vector<int> v(a, a+5);
list<int> l(v.begin()+1, v.end());
```
###earse()
- `earse(p)`删除**迭代器**p所指向数据。  
- `earse(p, q)`删除范围**[p, q)**内的所有数据。  

两种重载返回值都是指向所删除数据的后继数据的迭代器。  
如果已到末尾则返回`end()`。
###assign()
`assign(i, j)`将迭代器所指范围**[i, j)**内的数据（副本）替代容器内原有数据。 
**i，j不能是该容器的迭代器**
###clear()
清空容器内所有数据
