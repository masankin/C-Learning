指针与自增
=======
共有五种搭配(不包括括号括住自增的情况)情况，可分为两类。
##优先级
    () > ++ > *

##后向自增
有两种：
- *p++
- (*p)++

```c
j = *p++;
//等价于
j = *p;
p++;

j = (*p)++;
//等价于
j = *p;
(*p)++;
```
##前向自增
有三种：
- ++(*p)；
- ++*p;
- *++p;

`++*p`和`++(*p)`等价。
```c
j = ++(*p);
//等价于
(*p)++;
j = *p;

j = *++p;
//等价于
++p;
j = *p;
```
